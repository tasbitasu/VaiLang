📚 VaiLang Guide

            

                

                    
1. Program Structure & Output

                    
Program starts with VAI START and ends with VAI END. Use DEKHAW VAI to print values.

                    
VAI START
DEKHAW VAI "Hello Vai!" ;
VAI END

                

                

                    
2. Arithmetic Operators

                    
Arithmetic keywords: JOGAVAI (+), MINUSVAI (-), GUNAVAI (*), VAGAVAI (/), FOGAVAI (%).

                

                

                    
3. Control Flow

                    
JODI VAI (if), GHUR VAI (while), FATIVAI (break), BARAVAI (increment).
